# CodePilot AI v3.0
# Phase 19 — AI Workspace

---

# Objective

Transform CodePilot AI from a repository analysis platform into a persistent AI-powered software engineering workspace similar to Cursor, Windsurf and GitHub Copilot Workspace.

The AI should always understand:

- Current Repository
- Current File
- Current Function
- Current Selection
- Current Graph Node
- Previous AI Conversations

without asking the user repeatedly.

---

# Current Problem

Today the workflow is:

Open File

↓

Select Code

↓

Click AI Action

↓

Receive Response

↓

Everything resets

The AI has no memory of what the user was doing.
