# CodePilot AI v4.0 Roadmap

# Phase 20 — AI Patch Generation

---

# Objective

Transform CodePilot AI from an AI code reviewer into an AI software engineer capable of generating safe code modifications.

Unlike previous AI actions that only explain code, this phase introduces patch generation with full diff previews and user approval before any changes are applied.

The user must always review changes before they are written.

---

# User Review Required

> [!IMPORTANT]
> Every generated patch must be previewed before application.
>
> The AI must never directly overwrite repository files.
>
> Users should always be able to:
>
> - Review
> - Reject
> - Copy
> - Download
> - Apply
> - Undo
